# Client–Serveur TCP/UDP en C++

## Description
Projet de programmation réseau en C++ mettant en œuvre une communication client–serveur via les protocoles TCP et UDP.

Le système permet l’échange de messages entre client et serveur à travers la gestion des sockets et des ports réseau.

## Technologies utilisées
- C++
- Sockets BSD
- TCP
- UDP

## Concepts abordés
- Programmation réseau bas niveau
- Communication orientée connexion (TCP)
- Communication sans connexion (UDP)
- Manipulation des adresses IP et ports
- Gestion des erreurs réseau
